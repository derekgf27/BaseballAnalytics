# Coach Pitch Pad — Quick Reference

**On defense:** tap the pitch type (FB, SL, etc.) each pitch — **opposing lineup** on the left, **our pitcher's** stats on top.  
**On offense (we're hitting):** tap **Fastball**, **Off-speed**, or **Breaking ball** — **our lineup** on the left, **opposing pitcher's** stats on top.  
**Analyst on Record** handles balls, strikes, and PA results.

---

## Color guide

| Color | Meaning |
|-------|---------|
| **Red** | Blocked or failed — read it before tapping again |
| **Amber** | Warning (offline or full count) |
| **Green** | Success — action worked |
| **Sky blue** | Sync mismatch — check with analyst |
| **Gray** | Normal status (not an error) |

---

## Red errors — what to do

| Message | Fix |
|---------|-----|
| **3 strikes** — finish the PA in Record, or Undo / Reset AB | Analyst ends the at-bat on Record, or you Undo / Reset AB |
| **4 balls** — finish the PA in Record, or Undo / Reset AB | Same |
| **That pitch is already logged** — wait before tapping again | Check **This AB**; wait ~½ second between taps |
| **That pitch already has a type** — Undo if it was a mistake | Use **Undo** or verify the sequence |
| **(Error) + You appear offline** | Reconnect Wi‑Fi; retry |

---

## Top banners

| Banner | Meaning | Action |
|--------|---------|--------|
| **Offline** — logging may fail | No network | Reconnect before logging more |
| **Live pitch sync paused** | Realtime link dropped | Check Wi‑Fi; keep Record open — pad retries every few seconds. Undo / Reset still work |
| **Sync note** — batter not on opposing lineup | Record batter ≠ lineup card | Tell analyst to confirm batter / lineup on Record |
| **iPad landscape** (green, dismissible) | Layout tip | Rotate iPad; tap Dismiss |

---

## Normal gray messages (not errors)

| Message | Meaning |
|---------|---------|
| **We're hitting** — tap Fastball / Off-speed / Breaking ball below | Your team is batting — log opponent pitch mix |
| **Full count at 3 strikes** / **Four balls** (amber overlay) | AB is done on count; analyst must finish PA on Record |
| **No pitches yet — tap a type below** | Ready on defense |
| **No pitches yet — tap Fastball, Off-speed, or Breaking ball below** | Ready on offense |
| **Set the batter / pitcher on Record** | Record not synced yet |

---

## Green toasts (brief confirmation)

- **Removed · [pitch type]** — Undo worked  
- **At-bat sequence cleared** — Reset AB worked  

---

## Buttons

| Button | Use |
|--------|-----|
| **Undo** | Remove last pitch for this batter |
| **Reset AB** | Clear all pitches this AB (asks to confirm — cannot undo) |

---

## Before opening the pad

- Analyst created today’s game and is on **Record** for the same game  
- iPad on **Wi‑Fi**, **landscape**  
- Pick the correct game on the pitch pad picker  

---

## One-line rule

**Coach iPad = pitch types on defense; Fastball / Off-speed / Breaking ball on offense. Analyst Record = the game.**
